🧮 Calculadora Simples (React + TypeScript)

Uma calculadora simples desenvolvida com React e TypeScript, com operações básicas e interface moderna.

🚀 Tecnologias utilizadas
React
TypeScript
Styled Components
JavaScript (ES6+)
✨ Funcionalidades
➕ Soma
➖ Subtração
✖️ Multiplicação
➗ Divisão
🧹 Limpar display
🔢 Entrada de números
⚠️ Tratamento de divisão por zero
📸 Preview

Interface simples com botões numéricos e operações básicas.

⚙️ Como rodar o projeto
1. Clone o repositório
git clone https://github.com/Poweur/calculadora.git
2. Acesse a pasta
cd calculadora
3. Instale as dependências
yarn

ou

npm install
4. Rode o projeto
yarn dev

ou (se for CRA)

yarn start
